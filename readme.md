# E-Shop Module

Test task to make e-shop Product's Attributes module.

There is a ERR-diagram in the root of the project (ERR-diagram.jpg).

### Used

*  [Laravel] - php framework
*  Bootstrap
*  JQuery

### Installation

```sh
$ git clone
  copy .env
$ composer install
$ php artisan key:generate
  setup DB in .env
$ php artisan migrate --seed
$ npm install
$ npm run dev 
```
